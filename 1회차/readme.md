## 전역 속성 global Attributes
모든 html 태그에서 공통으로 사용할 수 있는 속성
- id : 고유값
- class : 두개 이상의 값 지정 가능, 다양한 태그에 중복적으로 사용 할 수 있다.
- style : CSS 속성 부여
- title : 말풍선

https://developer.mozilla.org/ko/docs/Web/HTML/Global_attributes
